总结

先用一张图总结一下所有常见的排序算法。

分类

说明一下跟排序相关的一些概念：

稳定性：
排序前后两个相等的数相对位置不变，则算法稳定。

为什么要定义稳定性，参考这样一个例子：
假设我们有一个 (student, class) 的序列

  1. (Dave, A)
  2. (Alice, B)
  3. (Ken, A)
  4. (Eric, B)
  5. (Carol, A)

想要对其进行名字和班级进行排序。
可以先对名字进行排序，得到

  1. (Alice, B)
  2. (Carol, A)
  3. (Dave, A)
  4. (Eric, B)
  5. (Ken, A)

再对班级进行排序，如果排序算法是稳定的，才一定可以得到：

  1. (Carol, A)
  2. (Dave, A)
  3. (Ken, A)
  4. (Alice, B)
  5. (Eric, B)

In-place 与 Out-palce：
即原地排序与非原地排序，区别在于是否需要额外空间。

内部与外部性：

  • 内部排序：排序过程中所有元素都在内存中。
  • 外部排序：排序过程中元素要在内外存中交换。

说明

    public static void swapArr(int[] arr, int i, int j){
        int temp = arr[i];
        arr[i] = arr[j];
        arr[j] = temp;
    }

插入类

插排将序列分为两个部分，有序序列和无序的序列。然后从无序序列中拿出一个元素在有序序列中进行比较再插入到有序序列中。

插入排序

注意和选择排序的区别，选择排序插入时不会在有序序列中进行比较，而插入排序会。

    public static int[] InsertSort(int[] unSortedArr) {
        int[] arr = Arrays.copyOf(unSortedArr, unSortedArr.length);
        // 表示需要插入的元素，长度为 1 的序列视为有序，所以进行 N-1 躺插入
        for (int i=1; i<arr.length; i++) {
            // 拿出要插入的元素
            int temp = arr[i];
            // 在有序序列中找到要插入的位置
            for (int j=i; j>0; j--){
                // 在遍历过程中将元素向后移动，空出位置给要插入的元素
                if(temp < arr[j-1]){
                    arr[j] = arr[j-1];
                }
            }
        }
        return arr;
    }

希尔排序

希尔排序将序列按照一定的长度切分为多个子序列，对子序列进行插入排序，再对整个序列进行插入排序。
即插入排序是 gap 为 1 的希尔排序的特殊情况。

    public static int[] shellSort(int[] unSortedArr) {
        int[] arr = Arrays.copyOf(unSortedArr, unSortedArr.length);
        // 取子序列时的相隔长度
        int gap = 1;
        while (gap < arr.length) {
            gap = gap * 3 + 1;
        }

        while (gap > 0) {
            // 对分组进行排序的过程
            for (int i = gap; i < arr.length; i++) {
                int tmp = arr[i];
                int j = i - gap;
                // i j 即为插入排序中的需要比较的元素
                // 这里就是插入排序的过程了
                while (j >= 0 && arr[j] > tmp) {
                    arr[j + gap] = arr[j];
                    j -= gap;
                }
                arr[j + gap] = tmp;
            }
            gap /= 3;
        }
        return arr;
    }

交换类

通过不断把大的值交换到小的值后面来完成排序。

冒泡排序

冒泡排序每次对比相邻的值进行交换，一直持续到序列尾。

    public static int[] bubbleSort(int[] unSortedArr) {
        int[] arr = Arrays.copyOf(unSortedArr, unSortedArr.length);
        // 最多发生 n-1 趟交换
        for (int i=0; i<arr.length-1; i++){
            // 用于标记是否发生了交换，如果没有发生交换说明已经有序
            boolean swapHappened = false;
            // 每趟交换交换相邻位置的元素，把大的元素向后冒泡
            for (int j=0; j<arr.length-1; j++){
                if (arr[j]>arr[j+1]){
                    swapArr(arr, j, j+1);
                    swapHappened = true;
                }
            }
            if (!swapHappened){
                return arr;
            }
        }
        return arr;
    }

快速排序

快速排序利用了归并思想，选取一个基准值，将基准值左右两侧的值与这个基准值比较，并交换，使得一边的值都大于或小于这个基准值，然后递归这个过程。

快速排序的关键之处在于切分，切分的同时要进行比较和移动。这里介绍两种切分方法。
单边扫描：
随意抽取一个数作为基准值，同时设定一个标记 mark 代表左边序列最右侧的下标位置，当然初始为 0 ，接下来遍历数组，如果元素大于基准值，无操作，继续遍历，如果元素小于基准值，则把 mark + 1 ，再将 mark 所在位置的元素和遍历到的元素交换位置，mark 这个位置存储的是比基准值小的数据，当遍历结束后，将基准值与 mark 所在元素交换位置即可。

双边扫描：
随意抽取一个数作为基准值，然后从数组左右两边进行扫描，先从左往右找到一个大于基准值的元素，将下标指针记录下来，然后转到从右往左扫描，找到一个小于基准值的元素，交换这两个元素的位置，重复步骤，直到左右两个指针相遇，再将基准值与左侧最右边的元素交换。


    public static int[] quickSort(int[] unSortedArr) {
        int[] arr = Arrays.copyOf(unSortedArr, unSortedArr.length);
        quickSortHelper(arr, 0, arr.length - 1);
        return arr;
    }

    public static void quickSortHelper(int[] arr, int startIndex, int endIndex) {
        if (endIndex <= startIndex) {
            return;
        }
        // 切分过程
        int pivotIndex = partitionSingle(arr, startIndex, endIndex);
        quickSortHelper(arr, startIndex, pivotIndex - 1);
        quickSortHelper(arr, pivotIndex + 1, endIndex);
    }

    // 单边扫描，从最左边开始扫描交换
    public static int partitionSingle(int[] arr, int startIndex, int endIndex) {
        // 选取基准值
        int pivot = arr[startIndex];
        // 标记左序列最右的索引
        int mark = startIndex;
        // 与基准值比较和交换的过程
        for (int i = startIndex + 1; i <= endIndex; i++) {
            if(arr[i] < pivot) {
                mark ++;
                swapArr(arr, mark, i);
            }
        }
        // 将 pivot 交换到中间
        arr[startIndex] = arr[mark];
        arr[mark] = pivot;
        return mark;
    }

    // 双边扫描
    public static int partitionDouble(int[] arr, int startIndex, int endIndex) {
        int left = startIndex;
        int right = endIndex;
        int pivot = arr[startIndex];

        while (true) {
            // 从左向右扫描
            while (arr[left] <= pivot) {
                left++;
                if (left >= right) {
                    break;
                }
            }

            // 从右向左扫描
            while (pivot < arr[right]) {
                right--;
                if (left >= right) {
                    break;
                }
            }

            // 左右指针相遇
            if (left >= right) {
                break;
            }
            // 交换左右值
            swapArr(arr, left, right);
        }

        // 将基准值插入到到序列中间
        swapArr(arr, startIndex, right);
        return right;
    }

选择类

类似于打扑克牌，在无序的序列中比较出一个值直接插入到有序序列中。
注意和插入类排序的区别在于是在有序序列中比较还是在无序序列中比较。

选择排序

    public static int[] selectionSort(int[] unSortedArr) {
        int[] arr = Arrays.copyOf(unSortedArr, unSortedArr.length);
        // 要比较 N-1 次，每次比较都会使得前 n 个序列是有序的
        for (int i=0; i<arr.length-1; i++) {
            // 用于记录无序序列中最小的值的索引
            int minIndex = i;
            // 在无序序列中找到最小值的索引
            for (int j = i+1; j < arr.length; j++) {
                if (arr[j] < arr[minIndex]) {
                    minIndex = j;
                }
            }
            swapArr(arr, minIndex, i);
        }
        return arr;
    }

归并类

归并排序

    public static int[] mergeSort(int[] unSortedArr) {
        int[] arr = Arrays.copyOf(unSortedArr, unSortedArr.length);
        mergeSortHelper(arr, new int[arr.length], 0, arr.length-1);
        return arr;
    }

    public static void  mergeSortHelper(int[] arr, int[] tmpArr, int startIndex, int endIndex) {
        if (endIndex <= startIndex) {
            return;
        }
        int middleIndex = startIndex + (endIndex - startIndex) / 2;
        // 切分区间的过程
        mergeSortHelper(arr, tmpArr, startIndex, middleIndex);
        mergeSortHelper(arr, tmpArr, middleIndex + 1, endIndex);
        // 合并排序过程
        merge(arr, tmpArr, startIndex, middleIndex, endIndex);
    }

    public static void merge(int[] arr, int[] tmpArr, int startIndex, int middleIndex, int endIndex) {
        // 拷贝需要排序的临时数组
        for (int i = startIndex; i <= endIndex; i++) {
            tmpArr[i] = arr[i];
        }
        // 需要合并的两个区间的起始索引
        int left = startIndex;
        int right = middleIndex + 1;

        for (int i = startIndex; i <= endIndex; i++) {
            // 左边区间排完序的情况
            if (left > middleIndex) {
                arr[i] = tmpArr[right];
                right++;
            }
            // 右边区间排完序的情况
            else if (right > endIndex) {
                arr[i] = tmpArr[left];
                left++;
            }
            // 按左右大小进行插入
            else if (tmpArr[right] < tmpArr[left]) {
                arr[i] = tmpArr[right];
                right++;
            }
            else {
                arr[i] = tmpArr[left];
                left++;
            }
        }
    }
